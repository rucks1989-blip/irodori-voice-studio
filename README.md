# irodori voice studio

> **開発中のローカル音声生成ツールです。**
irodori voice studio

    Windows向けローカル音声生成ツール（開発中）

Irodori-TTS（audio.cpp版）をブラウザーから手軽に利用するためのWindows向けローカルUIです。
低スペックのGPU（VRAM 8GBレベル）でも動くように作っています。
文章の分割、話者切替、参照WAV管理、無音挿入、WAV結合までをまとめて行えます。

通常の音声生成にLLMは必要ありません。
特徴

    Windows向けローカルUI

    LLMなしで音声生成可能

    ChatGPT・Gemini・Claudeなどで作成した文章をそのまま音声化

    話者タグによる自動参照WAV切替

    Irodori-TTS向け長文自動分割

    分割プレビュー

    チャンク結合・無音挿入

    REST API対応

    GTX1070 8GB環境で動作確認済み（audio.cpp CUDA版）

話者タグに対応する参照WAVを登録しておくことで、自動的に切り替えて読み上げます。
ローカルLLMとの組み合わせ

本ソフトでは次のような運用を想定しています。

LLM起動
    ↓
文章生成
    ↓
LLM終了（VRAM解放）
    ↓
Irodori-TTSで音声生成

この運用により、LLMと音声生成でVRAMを共有できます。

実際にGTX1070（VRAM8GB）環境で、この方式によるローカルLLM＋音声生成を確認しています。

Irodori-TTS向け長文分割

Irodori-TTSは長時間の連続生成では品質が低下する場合があります。

本ソフトではその特性を考慮し、

    約120文字を目安

    句読点を優先

    不自然になりにくい位置

で自動分割します。

長文でも自然につながるよう設計しています。

## 初回セットアップ

1. このリポジトリをダウンロードまたは展開します。
2. `setup.bat` をダブルクリックします。
3. 表示されたライセンス、ダウンロード内容、PC環境への変更内容を確認します。
4. 同意できる場合のみセットアップを続行します。
5. セットアップ結果に未達成項目が表示された場合は、同時に作成される案内を確認します。
6. 準備が完了したら `start.bat` をダブルクリックします。

初回セットアップ後の起動は `start.bat`、終了は `stop.bat` を使用します。未セットアップの状態で `start.bat` を押した場合は、先にセットアップが案内されます。

セットアップが自動導入できない項目は `download_links.html`、詳しい補足は `SETUP_MANUAL.txt` に記載しています。


## 基本的な使い方

### 通常の音声生成

1. 「音声生成」画面へ文章を入力します。
2. 必要なら「今回使う参照WAV」からWAVを選択します。
3. 「音声を生成」を押します。
4. 「分割を確認」で読み上げ単位を確認します。
5. 生成後、画面上で再生するかWAVを保存します。

参照WAVを選択しない場合は、Irodori-TTS内部の音声が使用される場合があります。初回生成はモデルや参照音声の読み込みにより時間がかかることがあります。

### 話者切替

1. 「話者切替」で使用するWAVを登録します。
2. 検出文字とWAVの対応を追加します。
3. 原稿中に同じ検出文字を記述します。

例：

```text
【佐藤】こんにちは。
【花子】おはようございます。
```

`【佐藤】` を `佐藤.wav`、`【花子】` を `花子.wav` に対応付けると、検出位置から次の話者まで参照音声が切り替わります。検出文字自体は読み上げません。

### 読まない文字列

設定した文字列を音声生成前に原稿から取り除けます。作品独自の注釈や、読み上げたくないラベルなどに利用できます。

## オプションのキャラクター会話

「会話」タブはOpenAI互換Chat APIへ接続し、LLMの返答を通常の音声生成処理へ渡します。返答にも分割、参照WAV、無音挿入、WAV結合が適用されます。

動作確認用として `characters/lily.sample.json` を同梱しています。これはキャラクターJSONの記述例であり、音声やLLMモデルは含まれていません。自作JSONは画面から追加できます。

初期設定例：

- Chat API: `http://127.0.0.1:11438/v1/chat/completions`
- Health API: `http://127.0.0.1:11438/health`
- Model alias: `character-chat`

使用するLLMおよびGGUFのライセンス、安全性、必要スペックは利用者自身で確認してください。



## 音声利用上の注意

- 使用する権利または許可を持つ音声だけを参照WAVとして使用してください。
- 本人の許可がないなりすまし、詐欺、誤情報、嫌がらせなどに使用しないでください。
- 生成物を公開・販売する場合は、参照音声、モデル、素材、公開先の規約を個別に確認してください。
- 生成音声の品質や内容は必ず人間が確認してください。

## 開発状況
